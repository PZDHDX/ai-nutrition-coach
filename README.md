## 项目概述

AI食养专家是一个微信小程序风格的饮食建议原型，面向三高和慢病风险人群，帮助用户记录饮食、识别高盐高糖高脂风险，并获得低盐、稳糖、护脂的日常食养建议。

本项目是 MVP 产品界面原型，不接入真实 AI、相机或后端接口。页面中的营养、评分和风险标签均为本地模拟数据，适合用于产品演示、交互评审和前端实现参考。

本项目只提供饮食管理和食养建议，不做疾病诊断，不替代医生、营养师或药师的专业判断，也不建议用户自行停药、换药或调整药量。

## 核心功能

- 首页食养状态：展示控盐、稳糖、护脂三项分数，以及钠、添加糖、膳食纤维、饱和脂肪等重点指标。
- 拍照识别流程：模拟识别食物，并给出高盐、精制碳水、添加糖、饱和脂肪、优质蛋白、高纤维等食养标签。
- 饮食记录联动：确认添加后自动更新首页、历史记录和食养分析。
- 食养分析图表：展示热量进度、三高关注指标、三大营养素比例、餐次分布和最近 7 天趋势。
- AI食养问答：围绕外卖、晚餐、早餐稳糖、火锅等生活场景给出温和可执行的建议。
- 个人设置：支持维护基础信息、食养目标、饮食偏好、目标热量和运动频率。

## 快速安装与运行

克隆项目：

```bash
git clone https://github.com/PZDHDX/ai-shiyang-expert.git
cd ai-shiyang-expert
```

本项目无外部依赖，直接使用浏览器打开即可运行：

```bash
index.html
```

也可以使用任意静态服务运行：

```bash
npx serve .
```

## 示例截图/使用演示

截图占位：

- 首页：展示 AI食养专家品牌、控盐/稳糖/护脂评分和今日重点营养指标。
- 食养识别：模拟识别红烧肉盖饭、珍珠奶茶、清炒青菜、无糖豆浆等食物，并展示食养风险标签。
- 饮食记录：按日期和餐次查看历史记录，点击食物可查看钠、添加糖和风险提示。
- 食养分析：展示热量进度、三高关注指标、三大营养素比例、餐次分布和趋势分析。
- 食养问答：通过聊天式界面获取外卖、晚餐、早餐稳糖、火锅等场景建议。
- 我的：维护基础信息、食养目标、饮食偏好和运动频率。

演示流程占位：

1. 从首页点击「拍照识别食养风险」。
2. 点击「拍摄照片」或「从相册选择」触发模拟识别。
3. 查看识别出的食物、食养风险标签和替换建议。
4. 调整份量后选择餐次，并确认加入饮食记录。
5. 进入「记录」查看历史饮食和食养分析。
6. 进入「食养问答」获取生活化饮食建议。

## 贡献指南

欢迎通过 Issue 或 Pull Request 改进本项目。

建议贡献方向：

- 补充更多三高人群常见饮食场景和模拟食物数据。
- 优化移动端交互细节、可访问性和视觉层级。
- 增加截图、演示 GIF 或在线预览地址。
- 将静态原型迁移为真实微信小程序工程。
- 接入真实图像识别、营养数据库或后端存储服务。

提交 PR 前建议检查：

- 页面在手机竖屏尺寸下显示正常。
- 拍照识别、确认加入记录、食养分析、AI 问答和设置保存流程可正常交互。
- 新增文案保持专业、温和、鼓励，不制造健康焦虑。
- 不加入疾病诊断、治疗承诺、停药建议或替代医生判断的表述。
