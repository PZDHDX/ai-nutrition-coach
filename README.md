## 项目概述

一个微信小程序风格的 AI 调香应用原型，面向香水爱好者、独立调香探索者和香氛产品策划场景。用户可以在首页切换「香水配方 / 精油方案」双模式，基于原料库生成差异化版本，管理原料厂家、产地、余量和香气简介，并通过 AI 调香师对话了解香材搭配、留香扩散、精油使用和合规表达知识。

本项目是 MVP 产品界面原型，不接入真实 AI、OCR、香材数据库或后端接口，适合用于产品演示、交互评审和前端实现参考。

## 核心功能

- 双模式工作台：首页支持香水配方和精油方案切换，按不同业务字段生成方案。
- 专业 brief 面板：香水模式支持香调、场景、季节、浓度、扩散、留香、预算和避免香材；精油模式支持使用方式、体验方向、基底/扩香和浓度提示。
- 原料库管理：记录原料名称、厂家、产地、批次、规格、剩余量、香气标签、推荐层级和简介。
- 专业选料抽屉：当前配方的前调、中调、后调和精油组合可从原料库选择，也可自定义新增原料。
- 模拟拍照导入：模拟 OCR 识别原料标签或库存清单，确认后批量导入原料库。
- 真实换版：每次点击「换一版」会生成新的版本号、材料组合、比例和 AI 评语。
- 配方结构展示：香水展示前调、中调、后调；精油展示精油组合、比例、基底和安全提示。
- 我的配方：保存生成过的调香方案，支持查看详情和作为模板再次编辑。
- AI 调香师：通过聊天式界面解答香水结构、精油搭配、留香扩散和功效类合规表达问题。
- 个人偏好设置：维护昵称、调香经验、偏好香调、常用浓度、预算范围和避免香材。
- 移动端优先界面：采用微信小程序风格的 4 Tab 底部导航，适合手机竖屏浏览和演示。

## 快速安装与运行

本项目无外部依赖，直接使用浏览器打开即可运行。

```bash
git clone https://github.com/pengzhangdehanduxiu/ai-nutrition-coach.git
cd ai-nutrition-coach
```

然后打开：

```bash
index.html
```

也可以使用任意静态服务运行，例如：

```bash
npx serve .
```

## 示例截图/使用演示

截图占位：

- 首页：展示香水/精油双模式工作台、专业 brief、强度滑杆和当前配方。
- 原料库：展示原料统计、库存状态、厂家、产地、剩余量、香气简介，并支持拍照导入。
- AI 调香师：通过对话解答调香知识、精油使用建议和合规表达问题。
- 我的：展示我的配方和个人调香偏好。

演示流程：

1. 从首页选择「香水配方」或「精油方案」。
2. 填写 brief，调整强度，点击「生成配方」。
3. 点击「换一版」查看不同材料、比例和版本号。
4. 点击前调/中调/后调的「选料」，从原料库选择或自定义新增原料。
5. 进入「原料库」查看库存，或使用「拍照导入」模拟识别原料清单。
6. 点击「保存配方」后，在「我的配方」中查看，或载入为模板继续调整。
7. 进入「AI调香师」询问香材搭配、库存替换、精油使用、留香扩散和合规表达问题。

## 贡献指南

欢迎提交 Issue 和 Pull Request 来改进这个原型。

建议贡献方向：

- 补充更多香调、精油、香材和配方模板。
- 优化移动端交互细节和可访问性。
- 增加截图、演示 GIF 或在线预览地址。
- 将静态原型迁移为真实微信小程序工程。
- 接入真实 AI 对话、香材数据库或用户配方存储服务。

提交 PR 前建议检查：

- 页面在手机竖屏尺寸下显示正常。
- 香水/精油模式切换、换版、选料抽屉、原料库导入、保存配方、我的配方、AI 问答和偏好保存流程可正常交互。
- 新增文案保持专业、克制、易懂，不夸大产品能力。
- 不加入未经验证的商业或功效承诺。
