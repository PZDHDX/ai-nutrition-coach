## 项目概述

一个微信小程序风格的 AI 智能营养管理应用原型，面向关注健康饮食、减脂、增肌、控糖和维持体重的人群。项目使用本地模拟数据演示拍照识别食物、每日饮食记录、营养分析、AI 营养师建议和个人目标设置等核心流程。

本项目是 MVP 产品界面原型，不接入真实 AI、相机或后端接口，适合用于产品演示、交互评审和前端实现参考。

## 核心功能

- 首页健康概览：展示今日摄入热量、目标热量、蛋白质、碳水、脂肪、饮水量和运动消耗。
- 拍照识别流程：模拟上传或拍摄食物照片，展示 AI 识别结果，并支持调整份量、删除食物和选择餐次。
- 饮食记录联动：确认添加后自动更新首页和历史记录页中的营养数据。
- 营养分析图表：展示热量摄入进度、三大营养素比例、餐次分布和最近 7 天热量趋势。
- AI 营养师聊天：内置快捷问题，结合用户目标、今日摄入和饮食记录生成温和可执行的建议。
- 个人目标设置：支持设置性别、年龄、身高、体重、目标体重、健康目标、饮食偏好、每日目标热量和运动频率。
- 移动端优先界面：采用微信小程序风格的 4 Tab 底部导航，适合手机竖屏浏览和演示。

## 快速安装与运行

本项目无外部依赖，直接使用浏览器打开即可运行。

```bash
git clone https://github.com/<your-name>/ai-nutrition-coach.git
cd ai-nutrition-coach
```

然后打开：

```bash
index.html
```

也可以使用任意静态服务运行：

```bash
npx serve .
```

## 示例截图/使用演示

截图占位：

- 首页：展示今日营养摄入概览和拍照识别主按钮。
- 拍照识别：模拟识别鸡胸肉沙拉、米饭、煎蛋、牛奶等食物，并支持调整份量。
- 饮食记录：按日期和餐次查看历史记录，点击食物可查看详情。
- 营养分析：展示热量进度、营养素比例、餐次分布和 7 天趋势。
- AI 营养师：通过聊天式界面获取个性化饮食建议。
- 我的：维护健康目标、基础信息和饮食偏好。

演示流程：

1. 从首页点击「拍照识别」。
2. 点击「拍摄照片」或「从相册选择」触发模拟识别。
3. 调整识别出的食物份量，选择加入早餐、午餐、晚餐或加餐。
4. 点击确认后返回首页，查看今日营养数据更新。
5. 进入「记录」查看历史饮食和营养分析。
6. 进入「AI营养师」获取基于目标和摄入记录的建议。

## 贡献指南

欢迎提交 Issue 和 Pull Request 来改进这个原型。

建议贡献方向：

- 补充更多真实饮食场景和模拟数据。
- 优化移动端交互细节和可访问性。
- 增加截图、演示 GIF 或在线预览地址。
- 将静态原型迁移为真实微信小程序工程。
- 接入真实图像识别、营养数据库或后端存储服务。

提交 PR 前建议检查：

- 页面在手机竖屏尺寸下显示正常。
- 拍照识别、确认加入记录、AI 建议和设置保存流程可正常交互。
- 新增文案保持专业、温和、鼓励，不制造健康焦虑。
- 不加入绝对化医疗承诺或治疗类表述。
