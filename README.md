## 项目概述

AI食养专家是一个微信小程序风格的饮食建议原型，面向三高和慢病风险人群，帮助用户记录饮食、识别高盐高糖高脂风险，并获得低盐、稳糖、护脂的日常食养建议。

本项目是 MVP 产品界面原型，不接入真实 AI、相机或后端接口。页面中的营养和风险数据均为本地模拟数据，仅用于产品演示、交互评审和前端实现参考。

## 产品边界

- 只提供饮食管理和食养建议，不做疾病诊断。
- 不替代医生、营养师或药师的专业判断。
- 不建议用户自行停药、换药或调整药量。
- 当用户涉及指标明显异常、严重不适、并发症、孕期、肾病、心衰或用药问题时，只提示咨询医生或专业人士。

## 核心功能

- 首页食养状态：展示控盐、稳糖、护脂三项分数，以及钠、添加糖、膳食纤维、饱和脂肪等重点指标。
- 拍照识别流程：模拟识别食物，并给出高盐、精制碳水、添加糖、饱和脂肪、优质蛋白、高纤维等食养标签。
- 饮食记录联动：确认添加后自动更新首页、历史记录和食养分析。
- 食养分析图表：展示热量进度、三高关注指标、三大营养素比例、餐次分布和最近 7 天趋势。
- AI食养问答：围绕外卖、晚餐、早餐稳糖、火锅等生活场景给出温和可执行的建议。
- 个人设置：支持维护基础信息、食养目标、饮食偏好、目标热量和运动频率。

## 快速运行

本项目无外部依赖，直接使用浏览器打开即可运行：

```bash
index.html
```

也可以使用任意静态服务运行：

```bash
npx serve .
```

## 验收建议

- 手机竖屏尺寸下首页、食养识别、饮食记录、食养分析、食养问答和我的页面显示正常。
- 高盐、高糖、高脂、精制碳水、优质蛋白、高纤维等标签能在识别结果和记录详情中展示。
- AI 回复不输出诊断、治疗承诺、停药建议或替代医生的判断。
- 设置页可多选控血压、稳血糖、护血脂，保存后首页和问答上下文同步更新。
